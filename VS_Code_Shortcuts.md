# Visual Studio Code Keyboard Shortcuts for Windows

## General

- `Ctrl+Shift+P` or `F1` - Show Command Palette
- `Ctrl+P` - Quick Open, Go to File
- `Ctrl+Shift+N` - New Window
- `Ctrl+Shift+W` - Close Window
- `Ctrl+,` - Open Settings
- `Ctrl+K Ctrl+S` - Keyboard Shortcuts
- `Ctrl+K Z` - Zen Mode
- `Ctrl+B` - Toggle Sidebar visibility
- `Ctrl+J` - Toggle Panel visibility
- `Ctrl+Shift+E` - Show Explorer / Focus Files
- `Ctrl+Shift+F` - Show Search
- `Ctrl+Shift+G` - Show Source Control
- `Ctrl+Shift+D` - Show Run and Debug
- `Ctrl+Shift+X` - Show Extensions

## File Management

- `Ctrl+N` - New File
- `Ctrl+O` - Open File
- `Ctrl+S` - Save
- `Ctrl+Shift+S` - Save As
- `Ctrl+K S` - Save All
- `Ctrl+F4` / `Ctrl+W` - Close Editor
- `Ctrl+K Ctrl+W` - Close All Editors
- `Ctrl+Tab` - Open Next Recently Used Editor
- `Ctrl+Shift+Tab` - Open Previous Recently Used Editor
- `Alt+F4` - Close Window

## Editing

- `Ctrl+X` - Cut line (empty selection)
- `Ctrl+C` - Copy line (empty selection)
- `Alt+Up` / `Alt+Down` - Move line up / down
- `Shift+Alt+Up` / `Shift+Alt+Down` - Copy line up / down
- `Ctrl+Shift+K` - Delete line
- `Ctrl+Enter` - Insert line below
- `Ctrl+Shift+Enter` - Insert line above
- `Ctrl+Shift+\` - Jump to matching bracket
- `Ctrl+]` / `Ctrl+[` - Indent / Outdent line
- `Ctrl+Shift+U` - Transform to uppercase (when configured)
- `Ctrl+U` - Soft undo
- `Ctrl+/` - Toggle line comment
- `Shift+Alt+A` - Toggle block comment
- `Ctrl+Shift+\` - Jump to matching bracket
- `Ctrl+Space` - Trigger suggestion
- `Ctrl+Shift+Space` - Trigger parameter hints
- `Ctrl+Shift+M` - Show Problems panel
- `Ctrl+.` - Quick Fix / Show Code Actions

## Navigation

- `Ctrl+G` - Go to Line
- `Ctrl+T` - Show All Symbols
- `Ctrl+Shift+O` - Go to Symbol in File
- `Ctrl+Shift+M` - Show Problems
- `F8` / `Shift+F8` - Go to Next / Previous Problem
- `Ctrl+Shift+\` - Jump to Matching Bracket
- `Ctrl+Home` / `Ctrl+End` - Go to Beginning / End of File
- `Ctrl+PageUp` / `Ctrl+PageDown` - Switch between open editors
- `Alt+Left` / `Alt+Right` - Navigate Back / Forward
- `Ctrl+Shift+[` / `Ctrl+Shift+]` - Fold / Unfold region
- `Ctrl+K Ctrl+[` / `Ctrl+K Ctrl+]` - Fold / Unfold all

## Search and Replace

- `Ctrl+F` - Find
- `F3` / `Shift+F3` - Find next / previous
- `Alt+Enter` - Select all occurrences of Find match
- `Ctrl+H` - Replace
- `Ctrl+Shift+H` - Replace in files
- `Ctrl+D` - Add selection to next Find match
- `Ctrl+F2` - Select all occurrences of current word
- `Ctrl+Shift+L` - Select all occurrences of current selection

## Multi-Cursor and Selection

- `Alt+Click` - Insert cursor
- `Ctrl+Alt+Down` / `Ctrl+Alt+Up` - Insert cursor below / above
- `Ctrl+Shift+L` - Select all occurrences of current selection
- `Ctrl+F2` - Select all occurrences of current word
- `Ctrl+D` - Add selection to next Find match
- `Ctrl+Shift+Alt+Right` / `Ctrl+Shift+Alt+Left` - Expand / Shrink selection
- `Shift+Alt+I` - Insert cursor at end of each selected line
- `Ctrl+L` - Select line
- `Ctrl+Shift+\` - Select bracket contents

## Editor Layout

- `Ctrl+\` - Split editor
- `Ctrl+1` / `Ctrl+2` / `Ctrl+3` - Focus first, second, or third editor group
- `Ctrl+K Ctrl+Left` / `Ctrl+K Ctrl+Right` - Move editor into previous / next group
- `Ctrl+K Z` - Toggle Zen Mode
- `Ctrl+Shift+` `[` / `Ctrl+Shift+` `]` - Focus previous / next editor group
- `Ctrl+K Ctrl+Shift+Left` / `Ctrl+K Ctrl+Shift+Right` - Focus group left / right

## Integrated Terminal

- `` Ctrl+` `` - Toggle integrated terminal
- `Ctrl+Shift+` `` - Create new terminal
- `Ctrl+C` - Copy selection in terminal
- `Ctrl+V` - Paste into terminal
- `Ctrl+Shift+C` - Copy path of active file
- `Ctrl+Shift+V` - Open Markdown preview to the side
- `Ctrl+Shift+` ` ` - Run active file in terminal (when configured)
- `Ctrl+PageUp` / `Ctrl+PageDown` - Switch terminal tabs
- `Ctrl+Shift+[` / `Ctrl+Shift+]` - Focus previous / next pane
- `Ctrl+Shift+` ` ` - Clear terminal (may require custom keybinding)

## Debugging

- `F9` - Toggle breakpoint
- `F5` - Start / Continue debugging
- `F11` - Step into
- `Shift+F11` - Step out
- `F10` - Step over
- `Ctrl+F5` - Run without debugging
- `Shift+F5` - Stop debugging
- `Ctrl+Shift+D` - Open Run and Debug view
- `Ctrl+K Ctrl+I` - Show hover
- `Ctrl+Alt+B` - Toggle breakpoints
- `Ctrl+Shift+F5` - Restart debugging

## Source Control

- `Ctrl+Shift+G` - Open Source Control view
- `Ctrl+Enter` - Commit in Source Control message box
- `Ctrl+Shift+K` - Discard changes
- `Ctrl+Shift+P` then type `Git: ...` - Use Git commands from Command Palette

## Window Management

- `Ctrl+Shift+N` - New Window
- `Ctrl+Shift+W` - Close Window
- `Ctrl+K F` - Close folder
- `Alt+F4` - Exit
- `Ctrl+Shift+T` - Reopen closed editor

## UI and Panels

- `Ctrl+Shift+U` - Show Output panel
- `Ctrl+` `` ` `` - Toggle integrated terminal
- `Ctrl+Shift+Y` - Show Debug Console
- `Ctrl+K M` - Change editor language mode
- `Ctrl+K V` - Open preview markdown
- `Ctrl+Shift+V` - Open Markdown preview

## Accessibility

- `Ctrl+M` - Toggle Tab moves focus
- `F1` - Open Help
- `Ctrl+K Ctrl+X` - Show accessibility options
- `Ctrl+K Ctrl+W` - Close all editors

## Notes

- Shortcuts can be customized via `File > Preferences > Keyboard Shortcuts` or `Ctrl+K Ctrl+S`.
- Some default shortcuts may change with VS Code updates or installed extensions.
- Use `Ctrl+Shift+P` and search for a command if you are unsure of the exact keybinding.
